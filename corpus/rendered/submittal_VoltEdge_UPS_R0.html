VOLTEDGE POWER SYSTEMS COMPLIANCE MATRIX VoltEdge PX-1200

Clause-by-Clause Conformance Statement

Below is our conformance statement indicating compliance with each clause of Section 26 33 53 of the construction specifications.

Spec Clause Parameter / Requirement Conformance Claim Remarks / Evidence
26 33 53 Part 1.3.1 UPS-SPEC-REVISION NC Non-Conformant. Submittal transmittal cover sheet states the submittal is reviewed against spec revision 'Rev 2020', but the active project specification is 'Rev 2024'.
26 33 53 Part 1.4.1.C UPS-CERTIFICATION NC Non-Conformant. Vendor submittal states compliance with IEC 62040-2 (EMC requirements) but does not provide certification under IEC 62040-3 (performance and test requirements) as explicitly mandated.
26 33 53 Part 2.1.2.A UPS-REDUNDANCY NC Non-Conformant. Vendor submittal proposes N+1 module configuration (5 active, 1 standby) which complies with the Design Basis Report but violates Section 26 33 53 Part 2.1.2.A which mistakenly demands a minimum of N+2 redundancy.
26 33 53 Part 2.2.1.A UPS-CAPACITY C Conforms. UPS module capacity is 1200 kW, which meets the specified minimum of 1200 kW.
26 33 53 Part 2.2.1.B UPS-POWER-FACTOR C Conforms. UPS module has a unity power factor (1.0), matching the specified 1.0 PF.
26 33 53 Part 2.2.1.D UPS-INPUT-THD C Conforms. UPS input THD is 4.2% at full load, which meets the specified maximum of 5.0%.
26 33 53 Part 2.2.2.A UPS-EFF-100 C Conforms. UPS module double-conversion efficiency at 100% load is 96.2%, meeting the specified minimum of 96.0%.
26 33 53 Part 2.2.2.B UPS-EFF-50 NC Non-Conformant. Datasheet table claims 96.1% efficiency at 50% load, but footnote 3 on page 8 states 'Efficiency reduces to 95.1% under VFI mode when harmonic filters are active at 50% load', which is below the 96% spec requirement.
26 33 53 Part 2.2.2.B UPS-EFF-75 C Conforms. UPS module double-conversion efficiency at 75% load is 96.0%, meeting the specified minimum of 96.0%.
26 33 53 Part 2.2.5.A UPS-OVERLOAD-CAPACITY NC Non-Conformant. UPS module overload rating is claimed as 125% for 5 minutes. Spec requires 125% overload capability for a minimum of 10 minutes.
26 33 53 Part 2.2.8 UPS-BYPASS-TRANSFER-TIME NC Non-Conformant. Bypass static switch transfer time (spec required <= 4ms) is completely missing from the vendor datasheet parameter table.
26 33 53 Part 2.4.2 UPS-AIRFLOW-PATH NC Non-Conformant. Datasheet shows cooling airflow path as rear-exhaust. Spec requires a front-inlet and top-exhaust airflow configuration for hot/cold aisle containment.
26 33 53 Part 2.4.5.C UPS-ENCLOSURE-PAINT NC Non-Conformant. UPS enclosure paint finish is listed as RAL 9005 (Jet Black). Spec requires RAL 7035 (Light Grey) polyester powder coating.
